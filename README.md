# FlowersAppp

Настольное приложение «Цветочный магазин» реализовано полностью на PySide6/QML.

## Подготовка базы данных

```bash
python -m models.database_init --reset
```

Скрипт по умолчанию наполняет базу тестовыми данными. Если нужно только
создать структуру без наполнения, добавьте флаг `--no-seed`.

## Запуск приложения

```bash
python -m ui_qt.main_qt
```

Приложение содержит интерфейсы клиента, сборщика и администратора.
